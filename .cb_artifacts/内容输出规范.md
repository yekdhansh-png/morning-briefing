# 晨光股市早报 · 内容输出规范

> 本文档定义早报里**每一类自动生成内容**的写作标准——字数、风格、必写要素、禁用词、正反例。  
> 这是**给产品/编辑/审稿用的标尺**，不是技术文档。

---

## 一、整体内容主张

**面向人群**：A 股个人投资者，盘前 7:30 看完准备开盘。

**核心立场**：
1. **事实优先于观点** — 标题陈述事实，分析才用判断
2. **未计价的边际增量优于已发酵旧闻** — 早报的核心价值是"今日开盘之前还没被定价的影响"
3. **可解释优于精准** — 宁可写"或/有望/大概率"留余地，也不写"必涨/抄底"
4. **正面叙事优于自爆** — 入选理由只写亮点，不写跌幅、净流出、回调
5. **双版本并行** — 每段长解读同时产出"老手专业版"+"新手白话版"，用户切档位

---

## 二、要闻 TOP 3 · 内容标准

### 2.1 标题（title）

**字数**：40-55 字（约 2.5-3 行）。少于 40 字不合格。

**必须写**：
- 关键主体（谁）+ 关键动作（做了什么）+ 关键事实/数据（结果是多少 / 什么时间 / 什么对象）
- 剥掉"据 XX 报道"/"公告称"等冗余前缀

**严禁写（17 词黑名单，命中即不合格）**：

| 类型 | 禁用词 |
|---|---|
| 预期/情绪 | 预期升温 / 情绪迎修复 / 信心增强 / 风险偏好抬升 |
| 节奏副词 | 有望 / 或将 / 大概率 / 有望加速 / 有望受益 |
| 映射类 | 映射 A 股 / 传导至 / 利好 XX 板块 / 景气度持续 |
| 状态变化 | 拐点临近 / 估值修复 / 战略协作深化 / 催化加速 |
| 评价类 | 迎来窗口 / 打开空间 / 格局重塑 / 信号积极 |

**正反例**：

| ❌ 错误（含判断） | ✅ 正确（纯事实） |
|---|---|
| 中美磋商释放积极信号，外贸链情绪迎修复窗口 | 中美磋商达成阶段性共识，双方原则同意对等降税 |
| 美股光通信逆势走强，AI 算力链景气度持续映射 A 股光模块板块 | 美股光通信板块逆势走强，Astera Labs 单日涨 13% |
| 普京下周访华，中俄能源合作战略协作深化预期升温 | 普京确认下周访华出席上合峰会，中俄拟讨论能源合作 |

**避免冗余**：早报顶部已展示外盘 6 大指数涨跌，海外类标题**避免重复**这些信息。若新闻同时含"三大指数涨跌"和"具体公司/板块事件"，把具体事件放在主位置。

---

### 2.2 核心信号（signal）

**字数**：60-80 字。

**必须三步式**：
1. **事件抽象**（约 15-20 字）— 把具体新闻压缩成市场状态变化（例："关税扰动转向缓和"）
2. **传导机制**（约 25-35 字）— 解释为什么影响 A 股（例："出口敏感品种估值压制有望解除"）
3. **资金映射**（约 20-25 字）— 翻译为资金流向（例："资金面或重新流入低位顺周期"）

**风格**：
- ✅ 节奏副词：或、有望、大概率、边际、优先
- ✅ 卖方术语：估值压制 / 比价映射 / 风险偏好抬升 / 估值锚 / 共振走强
- ❌ 绝对化：必涨 / 必跌 / 抄底 / 建议买入

---

### 2.3 信号详解（signalDetails）3 段

每段含：
- **tag**（≤8 字标签）：如"关税路径"/"受益方向"/"农产品端"
- **content**（30-45 字详解）

**整体结构**：3 段不能并列重复，要有**递进或对照**关系。常见结构：
- 事件层 → 受益方向 → 风险点
- 政策层 → 资金层 → 板块层
- 海外层 → 国内层 → 龙头层

---

### 2.4 影响资产（affectedName）

**必须是 A 股可交易的概念板块/行业板块**，≤5 字。

**正例**：光通信 / 国产芯片 / 商业航天 / 机器人 / 稀土永磁 / AI 算力 / 白酒 / 电力 / 油气开采 / 油运 / 绿电运营 / 存储芯片

**禁止写**：

| 类型 | 例子 |
|---|---|
| 太宽泛 | 能源链 / 宏观经济 / 外贸链 / 消费 / 金融 / 新能源 / 半导体行业 |
| 资产类别 | 股市 / 债市 / 商品 |
| 国家/事件 | 中俄合作 / 美联储 / 中美关系 |
| 后缀冗余 | 任何带"XX 板块"后缀的写法 |

**正反对照**：

| 新闻 | ❌ 错 | ✅ 对 |
|---|---|---|
| 英伟达 Q1 财报超预期 | 半导体行业 | AI 算力 |
| 中俄延续条约 + 能源合作 | 能源链 | 油气开采 / 油运 |
| 发改委升级绿电直连 | 新能源 | 绿电运营 |
| 稀土出口管制升级 | 有色 | 稀土永磁 |
| DRAM 内存连涨 | 半导体 | 存储芯片 |

---

### 2.5 影响方向（affectedDirection）

二选一：**正面 / 负面**。

---

### 2.6 影响强度（strength）

三选一：**强 / 中 / 弱**。

判断标准：
- **强**：S 级文件落地 / 龙头业绩超 30% / 单日成交额翻倍
- **中**：政策细则但无具体金额 / 业绩符合预期 / 板块普涨
- **弱**：意见稿征求 / 单一公司事件 / 边缘题材

## 三、全球资产综合解读 · 内容标准

### 字数与结构

**老手版**：80-100 字。  
**新手版**：100-130 字。

**必须三步式**：

| 段 | 字数 | 内容 |
|---|---|---|
| ① 整体格局抽象 | 15-25 字 | "股强 / 股弱 / 贵金属强 / 原油弱 / 风险偏好回升" 句式 |
| ② 驱动逻辑 | 30-45 字 | 从美股收评原文提取关键事实（美债/通胀/财报/地缘）解释"为什么" |
| ③ A 股映射 | 25-35 字 | 必须给**具体板块名**（AI 算力 / 光通信 / 出口链 / 有色） |

### 风格规则

| 老手版 | 新手版 |
|---|---|
| 用术语：风险偏好抬升、估值压制解除、共振走强、比价映射 | 翻译成大白话：'敢买股票的情绪在升温'、'压住股价的因素消失了'、'一起涨'、'国外公司涨了国内同行也容易涨' |
| 节奏副词：或、有望、大概率、边际 | 多用"就是说"、"简单讲"、"打个比方"、"相当于" |
| 不解释板块名 | 板块第一次出现要解释（"AI 算力（给 AI 做计算的服务器、芯片、机房等）"） |

### 严禁

- ❌ 罗列个股（"英伟达跌 0.77%"）
- ❌ 复述 6 资产数字（数字已经在卡片里了）
- ❌ 绝对化措辞、投资建议
- ❌ 凭空数字（原文没有的数据不能编）
- ❌ 泛泛说"利好市场"，必须给具体板块名

### 正反例

❌ 错误：  
"美股下跌，对 A 股形成压力，建议关注半导体板块。"

✅ 正确（老手版）：  
"全球呈现股弱、贵金属偏强、原油走弱的分化格局。美债 30 年收益率创 19 年新高压制风险偏好，但费城半导体逆势走平，AI 主题韧性仍在。对 A 股 AI 算力与光模块板块构成正向比价映射，石化与出口链短期承压。"

✅ 正确（新手版）：  
"昨夜全球市场分化：美股下跌、黄金白银涨、原油跌。原因是美国国债的利息涨到了 19 年最高水平——简单讲就是把钱存美元更划算了，所以大家不太愿意买美股。但做光通信（给 AI 服务器传数据的零部件）的公司逆市大涨，说明 AI 主题还很火。对 A 股来说，做 AI 服务器、光模块的公司今天可能跟着涨；做石油石化、做出口生意的公司可能承压。"

## 四、利好催化 TOP 2 · 内容标准

### 4.1 概念（concept）

**≤6 字**，必须是**可在 A 股交易的板块/概念名**。

正例：绿电 / 光通信 / 智能眼镜 / 国产芯片 / 商业航天 / 稀土永磁 / 机器人 / 创新药

---

### 4.2 标签（tag）封闭 6 选 1

LLM **不允许自创**，必须从下表选：

| 标签 | 适用 |
|---|---|
| **政策聚焦** | 国家/部委/地方政府发文、政策落地、试点扩容、监管动作、顶层会议表态、国家间合作 |
| **机构热评** | 券商研报、大行预测、机构调研、评级上调、机构观点集中表态 |
| **市场热议** | 雪球/论坛广泛关注、龙虎榜、资金抢筹、北向加仓 |
| **资产映射** | **仅限大宗商品价格**（金/银/铜/油/气/煤/铁矿/稀土的 LME/COMEX/NYMEX 价格变动） |
| **涨价信号** | 工业品/消费品价格上涨、供需缺口、紧平衡、库存告急、出口管制 |
| **景气赛道** | 订单饱满、出货激增、业绩兑现、周期反转、海外巨头新品/财报/发布会 |

**关键边界**：
- "谷歌发布会确认智能眼镜" → **景气赛道**（不是资产映射！）
- "英伟达财报超预期" → **景气赛道**（不是资产映射！）
- "LME 铜价创新高" → **资产映射**（大宗价格才算）
- "中俄能源合作" → **政策聚焦**（国家间合作算政策）

---

### 4.3 事件（event）

**≤25 字**，1 句话陈述事件本身，"谁 + 做了什么"。

例：
- "国家发改委升级多用户绿电直连政策"
- "谷歌确认智能眼镜 2026 秋季上市"
- "中央汇金增持四大行 ETF"

**严禁判断词**（同要闻 title 黑名单）。

---

### 4.4 催化路径（catalystPath）

**≤25 字**，用箭头串联"事件→机制→受益"三段。

例：
- "政策落地 → 绿电消纳价值重估 → 运营商估值修复"
- "谷歌发布会 → 智能眼镜量产预期 → 产业链受益"
- "美债收益率回落 → 风险偏好回升 → 成长股启动"

---

### 4.5 催化逻辑（catalyst）30-50 字

**必须明确点出信源关键词**：发文 / 公告 / 披露 / 发布会 / 研报。

如有基本面信号必须显式提及量化数据：'价格涨 X%' / '订单可见 Q3' / '同比 +X%'。

**老手版例**：  
"国家发改委、能源局联合发文升级绿电直连政策，叠加绿电消纳价格中枢上移，板块价值重估"

**新手版例**（40-60 字）：  
"国家发改委和能源局联合发了通知，让多家用户可以一起共建绿电项目，再加上卖电的价格能往上涨，相关公司股价应该重新被定价。"

---

### 4.6 看好理由（reason）40-60 字

**必须节奏副词**：有望 / 或 / 大概率 / 边际。

**严禁**：必涨 / 抄底 / 建议买入。

**老手版例**：  
"政策落地驱动板块情绪修复，叠加电力市场化改革大主线，绿电运营商估值或迎修复，新能源 ETF 大概率高开"

**新手版例**（50-75 字）：  
"政策落地后大家对绿电公司更有信心了，再加上电价改革大背景，做绿电生意的公司今天股价大概率往上走，相关 ETF 也容易高开。"

---

### 4.7 五层评估模型（决定哪些事件能入选）

| 层 | 维度 | 应用 |
|---|---|---|
| 1 | **去重** | 不能和要闻 TOP 3 重复（同主题不行，同概念也不行） |
| 2 | **热门度** | 主线赛道优先（AI 算力、机器人、商业航天、绿电…），冷门小众淘汰 |
| 3 | **时效（边际定价）** | 凌晨/盘后未计价 ★★★ > 昨日盘中 ★ > 24h+ 旧闻直接淘汰 |
| 4 | **信源等级** | S 部委文件 > A 龙头公告 > B 海外巨头 > C 机构观点 > D 传闻（基本不选） |
| 5 | **行业基本面信号** | 价格 / 供需 / 业绩 / 拐点 / 需求 — 含量化数据加分项 |

**硬约束**：2 条 TOP 必须分属不同赛道，且至少一条是 **S 级或 A 级**。

## 五、受益股 · 内容标准

### 5.1 选股两层逻辑

| 层 | 维度 | 标准 |
|---|---|---|
| **第 1 层（必）** | 直接受益 | 龙头股 / 业务占比高 / 产业链关键位置 |
| **第 2 层（加分）** | 股性活跃 | 适中市值（100-1500 亿）/ 主力 5 日净流入 / 历史强势属性 |

**搭配原则**：每条利好选 2 只 = **一龙头 + 一弹性**。

---

### 5.2 入选理由（reason）30-40 字

**必写**：
- **受益逻辑**：该股票为什么是本利好的核心受益方（业务/产品/客户/份额/地位）
- **角色定位**：龙头 / 弹性 / 次新（1-2 个词）

**选写**（亮点才写，没亮点宁可不写）：
- 涨跌幅：✅ 5/20/60 日涨幅**为正**且呈"温和上涨/突破/启动"特征 → 写
- 涨跌幅：❌ 大跌、净流出、回调 → **绝对不写**（自爆）
- 主力资金：✅ 5 日净**流入** → 写  
- 主力资金：❌ 净**流出**（哪怕轻微） → **绝对不写**

### 5.3 没正面数据时改用 5 类替代亮点

1. **业务地位**：全球龙头 / 国内市占率第一 / 客户独家供应
2. **业绩兑现**：已批量出货 / 订单可见 Q3 / 业绩同比大增
3. **产业地位**：核心代工商 / 关键材料供应商 / 唯一国产替代
4. **题材属性**：板块龙一 / 历史强势股 / 资金抱团对象
5. **估值与机构**：北向重仓 / 公募加仓 / 机构集中调研

---

### 5.4 reason 防自爆扫描（黑名单）

**绝不能出现的词**：净流出 / 流出 / 跌幅 / 下跌 / 回调

**绝不能出现的负数**：`-X%` / `-X 亿` / `-X 万`

### 5.5 正反对照

❌ 错误（含负面数据）：  
"风电运营弹性标的，20 日 +13.9%，60 日 +47.3%，**主力 5 日 -17.9 亿**，股性活跃。"

✅ 正确（替换为正面属性）：  
"国内风电运营第二梯队龙头，绿电交易弹性突出，板块情绪修复期估值修复空间大。"

❌ 错误：  
"AR 光波导核心供应商，**主力 5 日 -3.43 亿**，弹性强。"

✅ 正确：  
"AR 光波导核心供应商，深度绑定 Meta/谷歌客户，智能眼镜放量首选标的，弹性强。"

---

### 5.6 新手版 reason（reason_newbie）40-55 字

把术语翻译成大白话：

| 老手版 | 新手版 |
|---|---|
| 光模块龙头 | 做光通信器件的头部公司 |
| AI 算力链核心标的 | AI 服务器的核心供应商 |
| 弹性突出 | 股价波动大涨幅可能更大 |
| 估值 | 股价水平 |
| 业绩兑现度高 | 实际赚的钱跟之前预测的对得上 |
| 北向重仓 | 外资买得很多 |

## 六、双版本风格对照（核心翻译表）

| 专业术语（老手版） | 大白话（新手版） |
|---|---|
| 估值锚 | 价格的参考标准 |
| 边际定价 / 边际增量 | 还没被市场反应过的新影响 |
| 景气度 | 生意好不好 |
| 估值压制解除 | 压住股价的因素消失了 |
| 估值修复 | 股价应该涨上来 |
| 比价效应 / 比价映射 | 国外类似公司涨了国内同行也容易涨 |
| 资金面 | 炒股的钱 |
| 映射路径 | 会传到哪些股票上 |
| 催化 | 能让股价上涨的事情 |
| 风险偏好抬升 | 大家敢买股票的情绪在升温 |
| 风险偏好回落 | 大家不敢买股票的情绪在加剧 |
| 顺周期 | 经济好的时候业务跟着好的行业 |
| 共振走强 | 一起涨 |
| 情绪修复 | 大家又敢买了 |
| 板块 | 同类型公司 / 做 XX 的公司 |

## 七、跨模块通用红线

不论哪个模块，以下表述**全部不允许出现**：

| 类型 | 禁用 | 替代写法 |
|---|---|---|
| 投资建议 | 必涨 / 抄底 / 建议买入 / 重点配置 | 或迎修复 / 大概率高开 / 边际改善 |
| 绝对化 | 一定 / 必然 / 绝对 / 100% | 大概率 / 有望 / 或 |
| 凭空数字 | 编造原文没有的具体数字 | 只引用原文已有数据 |
| 罗列个股 | 在 signal/catalyst/summary 里堆个股代码 | 抽象成"龙头一线" / "板块" |
| 主观情绪 | 暴跌 / 崩盘 / 暴涨 / 飙升 | 走低 / 回调 / 走高 / 上行 |

## 八、版本切换规则

用户在底部对话窗选择**老手 / 新手**：
- 默认 **老手**（专业版）
- 切换 **即时生效**，无需刷新
- 同一份 briefing.json 同时携带双版本字段，前端按 `prefs.level` 切字段
- 输入框可用自然语言切换："切新手 / 我是新手 / 切老手"
